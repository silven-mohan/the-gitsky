# the-gitsky

Interactive main-page night sky built with ThreeJS on a modern Vite + React + TypeScript stack.

## Quick start

1. Install Node.js 20+.
2. Install dependencies:

	```bash
	npm install
	```

3. Start the dev server:

	```bash
	npm run dev
	```

4. Open the local URL shown in the terminal.
